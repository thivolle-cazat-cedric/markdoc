## Instatlation 

#### requirement

 * python2.7 ou +
 * virtualenv
 * pip


#### installation python 2.7

```
$ cd {{your-target-dir}}
$ ssh://git@baltig.thivolle-cazat.fr:2288/root/www-askcli.git
$ cd www-askcli
$ mkdir env
$ virtualenv python2
$ source env/python2/bin/activate
$ pip install -r requirement.txt
```
